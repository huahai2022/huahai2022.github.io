第一部分：linux的规则与安装

第〇章：计算机概论

0.1 电脑：辅助人脑的好工具

计算机就是接受用户输入的命令和数据，经由中央处理器的算术与逻辑单元运算处理后，产生或存储成有用的信息。如：加减乘除计算器、被印度阿三儿挖走的ATM机、全球定位系统GPS等等

0.1.1电脑硬件的五大单元

张三参加2022年高考，考完数学之后高兴坏了，就这破题还想阻止我进厂，真的是太逊了。于是打算去买一台电脑查一下哪个电子厂的薪水高，刚买到手，好奇的张三四处打量，嗷~电脑主要有三个部分：

  • 输入单元：键盘，鼠标，触摸屏幕，扫描仪，手写板，读卡器
  • 主机部分：系统单元，里面有一大堆板子，板子上有CPU，还有内存，适配卡
  • 输出单元：屏幕，打印机

张三打开电脑，百度一下：荣光电子厂，年薪过万不是梦，张三气坏了，搜了一天，电子厂月薪1100。一怒之下，张三把电脑拆了。果然里面有一大推板子，其中最重要的是一块主板，CPU安装在上面，上面还有内存、硬盘和一些其他的适配卡。与手机主板不同，手机主板上CPU和储存卡都是焊接在上面的。张三放弃了自己的”电子厂梦“，打算自学计算机成为”大牛“，毕竟是张三在哪里都会发光。

主机的重点就是CPU，全名中央处理器，英文名Central Processing Unit,是一个具有具体功能的芯片，里面有指令集，如果张三想要主机进行一些特殊服务，就要看看CPU有没有相关内置的指令集，毕竟礼貌的张三不能强人所难。CPU分为算术逻辑单元和控制单元两个主要的部分，算术逻辑单元负责负责程序运算和逻辑判断，控制单元负责协各个周边组件与个单元的工作。CPU负责运算和控制，CPU读取的数据从内存中来，内存中的数据从输入单元中得到，CPU处理完数据后先写回内存，再通过输出设备输出。如图所示

0.1.2一切设计的起点：CPU的架构

所有使用的软件都要经过CPU内部指令集来完成，根据两种设计理念被分为精简指令集（PISC）和复杂指令集（CISC）

  • 精简指令集（Reduced Instruction Set Compute):指令集较为精简，每个指令运行的时间很短，完成操作简单，指令性能佳，但完成复杂工作需要多条指令来完成。RISC指令集有：Oracle公司的SPARC、IBM公司的PowerArchitecture和ARM公司的ARM CPU等
  • 复杂指令集（Complex Instruction Set Computer):指令数目多且复杂，每条指令的长度不尽相同，指令执行复杂，每条指令执行的时间较长，但是指令处理的工作较为丰富。常见CISC指令集有：AMD、Intel、VIA等x86架构的CPU。

[^x86是指最早有Interl公司开发出代号为8086的CPU，后来以此框架开发出80286等CPU，这种架构CPU就被称为x86架构。x86架构CPU经历了从8位到16位再到32位，最后到64位的发展过程，因此现在的电脑多为64位，成为x86-64架构。位是CPU一次读取数据的最大值，64位表示CPU最大一次读取64位的数据，就是8个字节。]:

0.1.3其他单元的设备

其他单元的重要设备主要有：

  • 系统单元：CPU与内存及主板相关的组件。如主板上的网卡，磁盘阵列卡，显卡等
  • 存储单元：包括内存和辅助存储。如：硬盘、软盘、光盘、磁带。
  • 输入、输出单元：键盘、鼠标、体感设备、触摸屏，打印机、投影仪等

0.1.4运作流程

可以将电脑和人体进行类比：

  • CPU=脑子：通过脑袋来判断与控制身体各部分的活动，每个人做的事情不一样也对应了CPU指令集的差异。（所以只要智商高，用脑子来做CPU不无道理）
  • 内存=脑袋中正在思考的事情：脑袋接受外界刺激，进行分析，暂时存放的地方，主要用来给脑袋提供做出判断的信息。
  • 硬盘=脑袋中存放回忆的记忆区块：没必要立刻处理的事情存放于此，对应电脑上没有正在运行的部分。
  • 主板=神经系统：将重要的组件连接起来，手脚活动也都是通过神经传导到手脚。
  • 各项接口设备=人体与外界沟通的手脚、皮肤、眼睛等：是人体与外界互动的重要关键。
  • 显卡=脑袋中的影像：将来自眼睛的刺激成像后在脑袋中呈现，所以显卡产生的数据源也是CPU控制的。
  • 主机电源=心脏：所有的组件要运行必须要有足够的电力供给才行，对应人体中的心脏。

整部主机中最重要的部分就是就是CPU和内存，而CPU的数据统统来自内存，如果要由过去的经验来判断事情时，也要将经验（硬盘）挪到目前的记忆（内存）当中，再交给CPU进行判断。

0.1.5电脑的分类

依据电脑复杂度和计算能力，可分为5大类

  • 超级计算机：运算速度最快，维护成本高、使用费用高。用于国防军事、气象预测、太空科技。
  • 大型计算机：包含多个高速CPU，用来处理大量数据和复杂计算。用于大型企业的主机，全国性的证券交易所。
  • 迷你计算机：仍然保持着大型计算机支持多用户的特征，但不必放到大型的空调机房。用于科学研究、工程分析。
  • 工作站：针对特殊用途而设计的电脑。用于学术研究和工程分析
  • 微电脑：麻雀虽小五脏俱全。分为台式机和笔记本。

就性能而言，微电脑的性能已经够快了，甚至比工作站等级的电脑还要快，但是工作站的电脑稳定不宕机，运算准确。

0.1.6电脑上面常用的计算单位(容量、速度)

电脑的运算能力除了CPU指令集设计优劣之外，还是主要有速度来决定，存放在电脑中获得数据也是由单位的。

  • 容量单位：电脑对数据的判断主要依据有没有通电来记录信息，电脑只认识0和1，0/1的二进制单位我们称之为位，也叫做比特。1字节=8位。还有以下换算。
进位制 Kilo Mega Giga Tera Peta Exa Zetta
二进制 1024B（字节） 1024K 1024M 1024G 1024T 1024P 1024E
十进制 1000B（字节） 1000K 1024M 1000G 1000T 1000P 1000E
  • 速度单位：CPU的命令周期使用MHz或是GHz之类的单位，Hz就是“次每秒”。网络传输方面，使用bit为单位，如Mbit/s.

0.2个人电脑架构与相关设备组件

个人电脑通常是x86的架构，linux在早期发展时就是依据个人电脑发展的，所以了解一下x86架构的各个组件十分重要。x86架构主要由Interl和AMD两家公司制作，早期设计理念和架构有较大差距，但是相互学习对方长处的结果就是目前两者的架构已经十分类似了。

[^图像转载自https://blog.csdn.net/gaiazhang/article/details/73290419]:

  • 南桥：负责连接速度较慢的设备接口，包括硬盘、USB设备、网卡等
  • 北桥：负责连接速度较快的CPU、内存和显卡等

主板上设计的插口还有CPU、内存、显卡接口、SATA接口插槽

0.2.1执行脑袋运算与判断的CPU

主板上半部分的中央就是CPU插槽，目前主流CPU都是双核以上的架构，就是一块CPU封装当中嵌入了两个以上的运算内核，就是一个物理的CPU外壳，含有两个以上的CPU单位。不同的CPU型号上有着不同的引脚，能够搭配的主板芯片组也不同，主机升级时不能只考虑CPU还要考虑主板上支持CPU的型号。

  • CPU的工作频率：外频和倍频

    频率就是CPU每秒可以进行运算的工作次数，影响CPU的性能。厂商在CPU内部进行再加速，于是就有了所谓的外频和倍频，外频是CPU与外部组件进行数据传输时的速度；倍频则是CPU内部用来加速工作性能的一个倍数，两者相乘才是CPU的频率速度。

    因为所有的设备都得通过北桥进行连接，因此每个设备的工作频率应该相同。这样的话所有的数据都被北桥卡死了，北桥不可能比CPU快，所以将内存控制器整合到CPU内部，连接CPU内存、显卡控制器技术，Interl采用QPI和DMI技术；AMD采用Hyper Transport技术将CPU直接与内存、显卡等设备通讯，不需要外部连接芯片（故北桥已经没有了），只需要考虑倍频即整体的频率即可。

    [^不同CPU之间不能单纯地以频率来判断运算性能，因为每块CPU的指令集不同，架构也不同，可使用的二级缓存和运算机制也可能不同，每次频率能够进行的工作指令数也不相同，所以频率只能用来比较同款CPU的速度]:

  • 32位和64位的CPU与总线“位宽”

    CPU中的数据统统来自于内存，内存能提供给CPU的数据量大的时候，整体系统的性能也会更高，通过由CPU中的内存控制芯片与内存间的传输速度“前端总线速度”来说明内存能提供的数据量。

    内存的工作频率，由CPU中的内存控制器决定，如CPU内置的内存控制芯片对内存的工作频率最大可达1600MHz；每个时钟周期能够传输的数据量，大多为64位，这个64位就是“位宽”。因此CPU从内存中取得的最快宽带就是1600MHz*64bit=1600MHz * 8B=12.8GB/s

    CPU每次能够处理的数据量称为字长，依据CPU的设计有32位和64位，早期32位的CPU每次能够解析的数据量有限，因此从内存中传来的数据量就有限，导致32位CPU最多只能支持4GB的内存。

  • CPU的等级

    对不同CPU规范等级就有了i386,i586,i686等，某些程序注明是给x86-64的CPU使用时，就不要装在i686以下等级的电脑中，可能无法运行，不过在x86-64的硬件下可以安装i386的软件，这些东西向下具有兼容性。

  • 超线程（Hyper-Threading）

    CPU命令执行周期都很短，因此运算内核经常处于闲置状态，假想CPU同时执行两个程序来增加系统的性能。

    实现原理：在一个CPU内部将寄存器分为两组，让程序“同时竞争CPU的运算单元”，而非通过操作系统的多任务切换，因此i7级别的CPU只有四个物理内核，通过HT技术，操作系统可以检测到八个内核，并且可以让每个内核逻辑上分离，同时运行八个程序。

0.2.2内存

内存的频率和宽带；即是数据读入内存中的速度，个人电脑的内存主要组件为动态随机存取内存（Dynamic Random Access Memory,DRAM)只有在通电时才能记录和使用，断电之后就消失，被称为挥发性内存。DRAM使用较为广泛的有SDRAM和DDR SDRAM两种，两者除了工作电压的不同，DDR采用了双倍数据传输速度，类似CPU的倍频。

内存的容量：内存容量也十分重要，因为所有的数据都要加载到内存中才能被CPU读取，如果容量不够大的话，会导致大容量数据无法完全加载，需要将暂时没有用到的数据进行清理才能进行数据加载。对服务器而言，内存的容量比CPU的速度还要重要。

  • 多通道设计

    因为所有的数据都必须存放在内存中，所以内存的数据位宽越大越好，传统主线位宽为64位，为加大位宽，将两个内存集合在一起，这就是双通道设计。服务器要求的内存速度更快，中级服务器经常提供三通道，甚至四通道的内存环境，如E5-2650 v3可以接受的最大通道数就是四通道且为DDR4。

  • DRAM和SRAM

    通过二级缓存将很常用的程序或者数据放置到CPU的内部，提升性能，二级缓存整合在CPU的内部，L内存的速度必须与CPU的频率相同，而DRAM无法达到这个速度，需要用SRAM（Static Random Access Memory）静态随机存取内存。

  • 只读存储器（ROM）

    主板上的显卡和内置网卡，该功能是否要启动以及启动时的参数被记录在一个叫做CMOS的芯片上，该芯片需要而歪歪的电源作为记录，所以主板上会有一个纽扣电池。类似CPU和内存，该组件的参数也是可以调整的，那么CMOS上的数据如何读取更新呢？开机时按下【DEL】，你会进入一个叫做BIOS的界面，这个程序是写死在主板上面的一个储存芯片之中，该芯片在没有通电的时候也能记录数据，这就是只读存储器，是一种非易失的存储。

    固件就是固定在硬件上面的控制软件，很多时候也是通过ROM来进行软件的写入。如BIOS程序就是一块硬件，控制着启动时各项硬件参数的获取；由此可见BIOS对电脑十分重要，随电脑的发展，BIOS也要进行变化，但是使用ROM写入的BIOS无法变化，所以现在多用FLASH ROM来写入BIOS。

0.2.3显卡

显卡英文名称VGA(Video Graphics Array),负责图形影响的显示扮演。因为图形影像的显示在于分辨率和颜色深度，而图像的显示会占用内存，因此显卡上有集成内存成为显存，显存容量影响屏幕分辨率和颜色深度。（显卡的显存）

3D图像加速的发展

  • 早期3D游戏的运算任务有CPU来完成，但是CPU并不是针对图形加速而做的，且CPU平时已经相当忙碌了。
  • 显卡厂商加入一块GPU来进行运算，加速3D。
  • 现在，显卡主要通过GPU的控制芯片来与CPU、内存进行通讯，也是高速运行的组件，传输越快越好。（显卡的运算）

显卡规格发展经历了PCL、AGP、PCLe的发展，目前显卡多为PCLe，PCLe采用类似管道的加速理念，如PCLe x1宽带位250MB/s,而PCLe x16的宽带达到了32GB/s。

0.2.4硬盘与存储设备

电脑的存储设备：硬盘，软盘，MO，CD，DVD，磁带机，U盘等，其中最常见的当属硬盘。

  • 硬盘的物理组成：许许多多的光盘碟片、机械手臂、磁头与主轴马达。依据桌面和移动电脑分为2.5英寸和3.5英寸。

    工作原理：数据写在具有磁性的碟片上，通过在机械手臂上的磁头完成读写，通过主轴马达让碟片进行转动，完成读写的操作，单一碟片的容量有限，硬盘上会有两个以上的碟片。

  • 碟片上的数据

    扇区：在碟片同心圆上面切出一个个小区块，有机械手臂读写，这个小区快就是磁盘的最小物理存储单位，成为扇区；同一个同心圆的扇区组合成的圆成为磁道。并且一个磁盘里面可能含有多个碟片，所有碟片上的同一个磁道称为柱面。

    磁片读取方式：从外圈向内圈进行读写，因为外圈圆比较大，占用的面积比内圈多，因此外围会有着更多的扇区，数据写在外区，转一圈，数据读写量比内圈多，所以默认由外圈开始读写。

  • 传输接口

    为提升磁盘的传输速度，磁盘和主板的连接接口经过了多次改良有，如下

    • SATA接口

      左边的是数据线，对应插口为数据插口，右侧为电源线，对应插口为数据插槽。

      SATA的传输速度

      版本 宽带(Gbits/s) 速度（MB/s）
      SATA1.0 1.5 150
      SATA2.0 3 300
      SATA3.0 6 600

      因为SATA传输接口传输数据时，传输10位编码仅有8位进行数据，另外两位是检验用，所以上面的速度总是宽带的0.8倍。

    • SAS接口

      SAS又称为串行式SCSL接口（Serial Attached SCSL),接口速度比SATA快，连接SAS的硬盘碟片的转速和传输的速度都比SATA硬盘好，但是价格贵；并且个人电脑上一般没有SAS接口，需要外接转接卡才能支持。

      版本 宽带(Gbits/s) 速度（MB/s)
      SAS 1 3 300
      SAS 2 6 600
      SAS 3 12 1200

      [^可以看到SAS接口依然存在数据算法的关系，10位编码中有8位数据]:

    • USB接口

      USB是外接式磁盘接口，适用于磁盘是外接式的接口。USB的传输速率：

      版本 宽带（Mbits/s) 速度（MB/s)
      USB 1.0 12 1.5
      USB 2.0 480 60
      USB 3.0 5G 500
      USB 3.1 10G 1000

      [^由此可见USB接口没有数据检验的过程，但是USB传输数据的速率远小于理论速率]:

  • 固态硬盘（Solid State Disk,SSD)

    传统硬盘的缺陷：传统硬盘与要利用驱动马达来转动碟片，但是通常情况下没有连续读写，大部分情况下只是读写一大堆小文件，因此传统磁盘要转好多圈才能督导所有的数据；如果数据放置地比较零散，读写的速度延迟就会更高。

    固态硬盘发展：厂商拿闪存去制作成高容量的设备，利用SATA接口或SAS接口，外形像磁盘，但是闪存替换了磁头和碟片，被称为固态硬盘。

    固态硬盘的优势：通过闪存直接进行读写，降低数据延迟的同时还很省电。但早期固态硬盘因闪存写入次数的限制造成SSD的寿命只有两年，目前通过RAID的机制来防止SSD的损坏。

  • 选购和使用须知

    如果你想在电脑主机中增加一块硬盘，除了主板可接受的插槽接口还需要考虑什么呢？

    • HDD还是SSD：以SSD作为系统盘，数据存储放在HDD上，系统运行快速的同时，数据存储量也大

    • 容量：HDD已经有8TB的产品，目前SSD的多为128~256GB

    • 缓冲存储器：这个缓存将硬盘常用的数据存储起来，加速系统的读写能力

    • 转速：HDD硬盘利用主轴马达来驱动碟片写数据，转速越快越好

    • 使用须知：电脑通电之后，绝对不要移动主机，避免震动硬盘，导致硬盘数据错误；也要按正常方式关机

0.2.5扩展卡与接口

  • 扩展卡：服务器因为某些特殊的需求，使用主板之外的其他适配卡，所以主板上会留有多个扩展接口的插槽，通常为PCLe,因为它的速度最快，但是某些老旧的卡还需要使用，会留有一两个PCL插槽。

  • 接口：你想要加额外的扩展卡就需要选择接口了，对PCLe而言，个人电脑主板常见的是x16，一般中级服务器中大多为多个x8接口。

  • 多通道卡（PCLe x8）的卡安装在少通道插槽（PCLe x4)的可用行：CPU最大仅能支持16个PCLe 3.0的通道数。对于一块x16的扩展卡，你可以安装在一块x16的插槽上，安装在4块x4的插槽上，抑或是安装在一块x8插槽和两块x4插槽上。你也可以安装在一块x4的插槽上，但是这个卡的性能只剩下1/4。
